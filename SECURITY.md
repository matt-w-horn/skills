# Security Policy

## Reporting a vulnerability

Report vulnerabilities privately via [GitHub's private vulnerability
reporting](https://github.com/matt-w-horn/skills/security/advisories/new)
or by email to matt [at] matthorn [dot] io. Please don't open public issues
for security reports.

This is a personal project maintained on a best-effort basis. I'll
acknowledge reports as quickly as I can, usually within a few days.
